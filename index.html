Размещение на хостинге

Сайт размещён на сервисе github pages. Был создан новый репозиторий.

Размещение файлов сайта в репозитории
  • git init — создание локального репозитория
  • git add . — добавление всех новых файлов в текущей директории в отслеживанемые
  • git commit -m "commit text" — добавления комментария к изменениям
  • git remote add origin url_to_remote — связывание локального репозитория с удаленным
  • git push -u origin main — Публикация указанной ветки (main) в удаленный репозиторий. Ветка main является корневой для хостинга.
